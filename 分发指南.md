# Patent Hub 分发指南

## 如何打包分发给其他人

### 方式 1：快速打包（推荐）

1. **运行打包脚本**
   ```
   双击 "打包发布.bat"
   ```

2. **生成的文件夹**
   - 会自动创建 `Patent-Hub-Release` 文件夹
   - 包含所有必需文件（约 10-20 MB）

3. **压缩并分发**
   - 右键 `Patent-Hub-Release` 文件夹
   - 选择"发送到" → "压缩(zipped)文件夹"
   - 将生成的 ZIP 文件发送给其他人

### 方式 2：手动打包

如果自动打包失败，可以手动创建发布包：

**必需文件：**
```
Patent-Hub/
├── patent-hub.exe          # 可执行文件（从 target/release/ 复制）
├── templates/              # 模板文件夹（整个复制）
├── static/                 # 静态资源文件夹（整个复制）
├── .env                    # 配置文件（从 .env.example 复制并重命名）
├── 启动服务器.bat          # 启动脚本
├── 开机自启动.vbs          # 自启动脚本
├── 安装开机自启动.bat      # 安装自启动
├── 卸载开机自启动.bat      # 卸载自启动
├── 使用说明.txt            # 使用说明
└── README.md               # 详细文档
```

**可选文件：**
- `patent_hub.db` - 如果想分享已有的专利数据库

## 接收方使用步骤

### 第一次使用

1. **解压文件**
   - 解压 ZIP 到任意位置（如 `D:\Patent-Hub`）

2. **配置 API 密钥（重要！）**
   - 用记事本打开 `.env` 文件
   - 填入 SerpAPI 密钥：
     ```
     SERPAPI_KEY=你的密钥
     ```
   - 可选：填入 AI 密钥（用于智能分析）

3. **启动系统**
   - 双击 `启动服务器.bat`
   - 浏览器会自动打开

4. **开始使用**
   - 在搜索框输入关键词
   - 点击"检索"按钮

### 设置开机自启动（可选）

- 双击 `安装开机自启动.bat`
- 系统会在每次开机时自动启动

## 系统要求

### 最低要求
- **操作系统**: Windows 7 或更高版本
- **内存**: 512 MB RAM
- **磁盘空间**: 50 MB
- **网络**: 需要联网（用于在线搜索）

### 推荐配置
- **操作系统**: Windows 10/11
- **内存**: 2 GB RAM
- **磁盘空间**: 200 MB（包含数据库）
- **网络**: 宽带连接

## API 密钥获取

### SerpAPI（必需，用于在线搜索）

1. 访问 https://serpapi.com/
2. 注册账号（免费版每月 100 次搜索）
3. 在 Dashboard 找到 API Key
4. 复制到 `.env` 文件的 `SERPAPI_KEY=`

### 智谱 AI（可选，用于 AI 分析）

1. 访问 https://open.bigmodel.cn/
2. 注册账号
3. 创建 API Key
4. 复制到 `.env` 文件的 `AI_API_KEY=`

## 常见问题

### Q1: 双击启动脚本没反应？
**A:** 
- 检查是否有杀毒软件拦截
- 右键 → 以管理员身份运行
- 检查 `patent-hub.exe` 是否存在

### Q2: 浏览器打不开？
**A:**
- 手动访问 http://127.0.0.1:3000
- 检查防火墙是否拦截
- 确认服务器已启动（命令行窗口显示 "Patent Hub running"）

### Q3: 搜索没有结果？
**A:**
- 检查 `.env` 文件是否配置了 `SERPAPI_KEY`
- 确认网络连接正常
- 尝试切换到"本地数据库"模式

### Q4: 可以在多台电脑上使用吗？
**A:** 
- 可以！每台电脑都需要：
  1. 复制完整的文件夹
  2. 配置自己的 API 密钥
  3. 双击启动脚本

### Q5: 如何更新版本？
**A:**
- 下载新版本的 ZIP
- 解压到新位置
- 复制旧版本的 `.env` 文件（保留配置）
- 可选：复制 `patent_hub.db`（保留数据）

### Q6: 如何卸载？
**A:**
1. 运行 `卸载开机自启动.bat`（如果安装了自启动）
2. 关闭服务器（关闭命令行窗口）
3. 删除整个文件夹

## 分发注意事项

### ✅ 可以做的
- 分享给同事、朋友使用
- 在多台电脑上安装
- 修改配置文件
- 自定义样式（修改 `static/style.css`）

### ❌ 不要做的
- 不要分享你的 API 密钥
- 不要用于商业用途（除非获得授权）
- 不要修改可执行文件
