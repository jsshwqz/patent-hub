## 快速开始

### 1. 下载安装

1. 访问 [GitHub Releases](https://github.com/jsshwqz/patent-hub/releases/latest)
2. 下载 `patent-hub-v0.1.0-windows-x86_64.zip`
3. 解压到任意目录（建议：`D:\patent-hub`）

### 2. 快速配置（推荐）

双击运行 `国内用户配置.bat`，按照提示配置：
- 智谱 GLM API Key（必需）
- SerpAPI Key（可选）

### 3. 启动使用

双击 `start.bat` 启动应用，浏览器会自动打开。

### 📡 代理配置方法

#### 方法 1：使用系统代理（推荐）

如果您使用 Clash、V2Ray 等代理工具：

1. 开启代理工具的"系统代理"模式
2. 直接启动 Patent Hub
3. 系统会自动使用代理

#### 方法 2：手动设置代理

```bash
# 设置代理（根据您的代理端口修改）
set HTTP_PROXY=http://127.0.0.1:7890
set HTTPS_PROXY=http://127.0.0.1:7890

# 启动应用
patent-hub.exe
```

#### 方法 3：修改启动脚本

编辑 `start.bat`，在开头添加：

```batch
@echo off
set HTTP_PROXY=http://127.0.0.1:7890
set HTTPS_PROXY=http://127.0.0.1:7890
patent-hub.exe
```

### 🚫 无代理使用方案

即使没有代理，您仍然可以：

1. **使用 AI 功能**
   - AI 分析本地专利

## API 配置

### 智谱 GLM（强烈推荐）

#### 为什么选择智谱 GLM？

- ✅ **国内直连**：无需代理，访问速度快
- ✅ **中文优化**：专为中文场景优化，理解能力强
- ✅ **价格实惠**：比 OpenAI 便宜很多
- ✅ **免费额度**：新用户赠送免费额度
- ✅ **稳定可靠**：清华大学技术支持

#### 注册步骤

1. **访问官网**
   ```
   https://open.bigmodel.cn/
   ```

2. **注册账号**
   - 支持手机号注册
   - 支持微信扫码登录
   - 完成实名认证（可选）

3. **创建 API Key**
   - 进入控制台
   - 点击"API 密钥"
   - 点击"创建新的 API Key"
   - 复制保存密钥（只显示一次）

4. **配置到 Patent Hub**
   
   编辑 `.env` 文件：
   ```env
   AI_API_KEY=你的智谱GLM密钥
   AI_BASE_URL=https://open.bigmodel.cn/api/paas/v4
   AI_MODEL=glm-4-flash
   ```

#### 模型选择

| 模型 | 速度 | 质量 | 价格 | 推荐场景 |
|------|------|------|------|---------|
| glm-4-flash | ⚡⚡⚡ | ⭐⭐⭐ | 💰 | 日常使用（推荐） |
| glm-4 | ⚡⚡ | ⭐⭐⭐⭐ | 💰💰 | 深度分析 |
| glm-4-plus | ⚡ | ⭐⭐⭐⭐⭐ | 💰💰💰 | 专业分析 |

**推荐配置**：`glm-4-flash`（速度快、质量好、价格低）

#### 费用说明

- **免费额度**：新用户赠送一定额度
- **按量计费**：用多少付多少
- **价格参考**：
  - glm-4-flash：约 ¥0.001/千字
  - glm-4：约 ¥0.01/千字
  - glm-4-plus：约 ¥0.05/千字

**示例**：分析 100 个专利（每个约 500 字），使用 glm-4-flash 约花费 ¥0.05

### SerpAPI（可选）

#### 注册步骤

1. **访问官网**
   ```
   https://serpapi.com/
   ```

2. **注册账号**
   - 支持 Google 登录
   - 支持邮箱注册

3. **获取 API Key**
   - 进入 Dashboard
   - 复制 API Key

4. **配置到 Patent Hub**
   
   编辑 `.env` 文件：
   ```env
   SERPAPI_KEY=你的SerpAPI密钥
   ```

#### 费用说明

- **免费额度**：100 次搜索/月
- **付费套餐**：
  - $50/月：5000 次搜索
  - $100/月：15000 次搜索

## 功能可用性

### ✅ 无需代理即可使用

**功能**：对专利进行深度分析

**使用方法**：
1. 在专利详情页点击"AI 分析"

**使用方法**：
1. 在专利详情页点击"文件对比"
2. 上传 TXT 文件（技术方案、产品说明等）

**应用场景**：
- 侵权风险评估
- 技术方案对比
- 产品与专利对比

#### 4. 本地数据管理

**功能**：管理已保存的专利数据

**可用操作**：
- 查看搜索历史
- 浏览已保存专利
- 查看统计图表
- 导出 Excel
- 数据备份

### ⚠️ 需要代理才能使用

#### 1. 在线搜索

**功能**：搜索 Google Patents

**使用方法**：
1. 开启代理

## 常见问题

### Q1：没有代理可以使用吗？

**A**：可以！大部分功能都不需要代理：
- ✅ AI 分析（智谱 GLM）
- ✅ 专利对比
- ✅ 文件对比
- ✅ 查看本地数据
- ✅ 统计图表
- ✅ 导出 Excel

只有"在线搜索"功能需要代理。

### Q2：智谱 GLM 和 OpenAI 哪个好？

**A**：对于国内用户，强烈推荐智谱 GLM：

| 对比项 | 智谱 GLM | OpenAI |
|--------|----------|--------|
| 访问速度 | ⚡⚡⚡ 快 | ⚡ 慢（需代理） |
| 中文理解 | ⭐⭐⭐⭐⭐ 优秀 | ⭐⭐⭐⭐ 良好 |
| 价格 | 💰 便宜 | 💰💰💰 贵 |
| 稳定性 | ✅ 稳定 | ⚠️ 需代理 |

### Q3：如何批量搜索专利？

**A**：建议策略：
1. 有代理时集中搜索
2. 保存到本地数据库
3. 之后可离线使用
4. 定期同步更新

### Q4：数据保存在哪里？

**A**：所有数据保存在本地：
- 数据库文件：`patent_hub.db`
- 位置：程序运行目录
- 可以备份和迁移

### Q5：如何备份数据？

**A**：
1. 复制 `patent_hub.db` 文件
2. 保存到安全位置
3. 需要时复制回来即可

### Q6：可以在公司内网使用吗？

**A**：可以！
- AI 功能需要访问智谱 GLM（https://open.bigmodel.cn）
- 如果内网可以访问外网，完全没问题
- 如果完全隔离，只能使用本地数据功能

### Q7：手机可以使用吗？

**A**：可以！
1. 电脑和手机连接同一 WiFi
2. 启动程序后查看 IP 地址
3. 手机浏览器访问：`http://192.168.x.x:3000`
4. 添加到主屏幕，像 APP 一样使用

### Q8：如何提高搜索效率？

**A**：
1. 使用精确关键词
2. 设置合理的日期范围
3. 批量保存结果
4. 利用本地数据库

## 最佳实践

### 1. 推荐配置

```env
# 使用智谱 GLM（国内最优选择）
AI_API_KEY=你的智谱GLM密钥
AI_BASE_URL=https://open.bigmodel.cn/api/paas/v4
AI_MODEL=glm-4-flash

# SerpAPI（可选）
SERPAPI_KEY=你的SerpAPI密钥
```

### 2. 使用流程

#### 有代理时：
```
1. 开启代理
2. 搜索专利
3. 批量保存
4. AI 分析
5. 导出数据
```

#### 无代理时：
```
1. 查看本地数据
2. AI 分析
3. 专利对比
4. 文件对比
5. 导出数据
```

### 3. 数据管理

- **定期备份**：每周备份 `patent_hub.db`
- **清理历史**：定期清理不需要的搜索历史
- **导出重要数据**：重要专利导出为 Excel 保存

### 4. 成本控制

- **使用 glm-4-flash**：性价比最高
- **批量分析**：一次性分析多个专利
- **缓存结果**：避免重复分析

### 5. 安全建议

- **保护 API Key**：不要分享给他人
- **定期更换**：建议每 3 个月更换一次
- **监控用量**：定期检查 API 使用情况
- **备份数据**：重要数据及时备份
